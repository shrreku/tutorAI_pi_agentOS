# Linear Algebra Mini Lesson

## Vectors

Vectors represent magnitude and direction. A vector can be written as an ordered list of numbers.

## Basis

A basis is a set of linearly independent vectors that spans a vector space. Every vector in the space can be written as a linear combination of basis vectors.

## Linear Combination

A linear combination is a weighted sum of vectors. Linear combinations depend on scalar multiplication and vector addition.

## Key Relationships

- A basis depends on linear independence.
- A basis depends on spanning the vector space.
- Linear combinations support vector construction.
- Standard basis vectors are an example of a basis.
